# Query Results and Business Interpretation

## Query 1: Top 10 Highest-Revenue Movies
This query identifies the movies with the highest box office revenue.  
It helps highlight the most commercially successful films in the dataset.

---

## Query 2: Top 10 Highest-Rated Movies
This query shows movies with the highest audience ratings.  
It reflects viewer satisfaction and perceived quality.

---

## Query 3: Average Rating by Year
This query calculates the average rating for each year.  
It helps analyze how audience perception changes over time.

---

## Query 4: Number of Movies by Year
This query counts how many movies appear per year.  
It shows dataset coverage and release trends.

---

## Query 5: Most Popular Movies
This query ranks movies by popularity score.  
It indicates which movies attract the most audience attention.

---

## Query 6: Top Distributors
This query shows which distributors appear most frequently.  
It helps identify dominant players in the dataset.

---

## Query 7: Average Revenue by Year
This query calculates average revenue per year.  
It helps identify trends in box office performance over time.
